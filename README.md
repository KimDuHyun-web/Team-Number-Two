## 프로젝트 개요

### 1.1 목표

**사이트 리뉴얼:** 이스트캠프 사이트의 장점과 단점을 분석 후 참조 사이트를 참고하여 보완

**협업 경험:** 수업 내용을 토대로 피그마와 웹표준에 맞춘 코딩 경험

### 1.2 팀원

| 이름 |  역할 | 주요 담당 |
|--- | --- | --- |
| 김두현 | 팀장·기획·디자인·코딩·회의록 정리 |   content - banner, faq, footer|
| 김진선 | 기획·디자인·코딩 |  header - nav, banner / content - edu, result |
| 조창희 | 코딩 | content - specail, benefit |

### 1.3 마일스톤

#### 1일차 - 기획
+ 현 사이트의 장·단점 분석
+ 타 사이트 벤치마킹

#### 2일차 - 디자인
+ 디자인 콘텐츠 구상
+ 디자인 파트 분배
+ 디자인 작업

#### 3일차 - 디자인, HTML 작성
+ 디자인 작업 마무리
+ HTML 파트 분배
+ 공통 HTML/CSS 작성
+ 맡은 파트 HTML 작성

#### 4일차 - HTML/CSS 작성
+ 피드백 수정
+ 맡은 파트 HTML/CSS 작성

#### 5일차 - HTML/CSS 작성, 발표 준비
+ 맡은 파트 HTML/CSS 마무리
+ 디자인과 비교 후 정리
+ 웹 접근성, 웹 표준 검사
+ 발표 슬라이드 제작
