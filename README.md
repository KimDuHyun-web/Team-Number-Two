## 4.제작후기

이 프로젝트로 배웠던 HTML과 CSS를 모두 사용해 볼 순 없었지만, 팀원 간 파트를 나누고 팀원들은 어떻게 HTML을 어떻게 구성하고 CSS를 작성하는지 볼 수 있었고, Git을 이용해 실제 협업을 경험하고 배울 수 있는 시간이었습니다. 
부족한 부분이 많아 어려웠지만 팀원의 도움을 받아 한 걸음씩 나아갈 수 있었고, 함께하는 프로젝트라 더 열심히 할 수 있어 더욱 뜻깊었던 프로젝트였습니다.
